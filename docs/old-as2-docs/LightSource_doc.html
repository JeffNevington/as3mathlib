com.wis.math.geom.util.LightSource class

version: 1.7

description:

Implements the behaviours of the LightSource Class.

Provides utlity methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:LightSource = new LightSource(ray,color);

parameters:

  • ray (Vector) -- a position Vector object.
  • color (Col) -- a Col object.
