com.wis.math.geom.intersect3D.Intersect3D class

version: 1.7

description:

Implements the behaviours of the Intersect3D Class.

Provides methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Intersect3D = new Intersect3D(objects);

parameters:

  • objects (Array) -- a list of shape objects.
