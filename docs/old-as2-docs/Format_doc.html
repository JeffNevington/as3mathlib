numCurrency

description: Unfinished method that introduces European formatting. I have included 'numCurrency2' that doesn't handle European formatting -- both versions are causing errors presently.

usage: 

Format.numCurrency(input,dig,bEuro,bCurr,sCurr);

parameters:

  • input (String) -- allows for direct textfield input.
  • dig (Number) -- a positive integer.
  • bEuro (Boolean) -- if true, reverses delimit_arr.
  • bCurr (Boolean) -- if false, returns a comma-delineated number.
  • sCurr (String) -- country currency symbol.

returns: (String) -- returns a string representation of the formatted currency.
