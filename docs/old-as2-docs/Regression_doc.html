com.wis.math.alg.Regression class

version: 1.6

description:

Implements the behaviours of the Regression Class as an extended MovieClip. It defines a class instance for polyline mapping, linear least squares, and polynomial least squares helper functions.

usage:

var inst:Regression = new Regression();

parameters:

  • none -- no input parameters.

plotPolyline

description: Sort x and y values by ascending x values and connect the points with line segments.

usage: 

inst.plotPolyline(x_arrmy_arr);

parameters:

  • x_arr (Array) -- data array containing x-values from the data set.

returns: (Void) -- if successful, a plot is generated on the current mc.

plotLinearData

description: Find the line (y = mx + b) which best fits a given set of points using the 'least squares' appoach: fit the line to the data such that the SUM of the SQUARES of the DIFFERENCES between the line and the data is as small as possible.

usage: 

inst.plotLinearData(x_arr,y_arr);

parameters:

  • x_arr (Array) -- data array containing x-values from the data set.
  • y_arr (Array) -- data array containing y-values from the data set.

returns: (Void) -- a plot is generated on the current mc.

plotPolynomialData

description: A method which implements n-degree polynomial solutions to fit a curve to a set of points. The 'least squares' appoach is employed: fit the curve to the data such that the SUM of the SQUARES of the DIFFERENCES between the curve and the data is as small as possible.

For degree = 2, the solution is quadratic: y = ax^2 + bx + c For degree = n, the solution is of the form: y = A0 + A1X + A2X^2 + A3X^3 + ... + AnX^n

usage: 

inst.plotPolynomialData(degree,x_arr,y_arr);

parameters:

  • degree (Number) -- one less than the number of terms in your polynomial.
  • x_arr (Array) -- data array containing x-values from the data set.
  • y_arr (Array) -- data array containing y-values from the data set.

returns: (Void) -- if successful, a plot is generated on the current mc, otherwise a trace() is fired.

calcLinearLeastSquares

description: Use linear least squares method to compute the m and b coefficients which best describe a given set of ordered pairs as a single line.

usage: 

inst.calcLinearLeastSquares(x_arr,y_arr);

parameters:

  • x_arr (Array) -- data array containing x-values from the data set.
  • y_arr (Array) -- data array containing y-values from the data set.

returns: (Object) -- a results object -- {m: m_term, b: b_term}.

calcPolynomialLeastSquares

description: Compute the a, b, c, ... n values for polynomial least squares, caller should check for a 'null' return value to indicate failure, otherwise an array of coefficients is returned.

usage: 

inst.calcPolynomialLeastSquares(degree,xVals,yVals);

parameters:

  • deg (Number) -- one less than the number of terms in your polynomial.
  • xVals (Array) -- data array containing x-values from the data set.
  • yVals (Array) -- data array containing y-values from the data set.

returns: (Array) -- failure returns null -- success returns an array of coefficients for the plotting equation.

returns: (Number) -- returns value of polynomial.

gaussEliminate

description: If possible, decompose a matrix of N (N + 1) coefficients into a single coefficient vector.

usage: 

inst.gaussEliminate(coeffs);

parameters:

  • coeffs (Array) -- an N by N+1 matrix representing the N+1 coefficients of N required equations.

returns: (Boolean) -- a boolean value is returned to indicate success or failure. Additionally, the "coeffs" array is modified directly for use by the calling function (result is the vector represented by column N+1 of 'coeffs').
