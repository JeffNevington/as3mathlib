com.wis.math.calc.solvers.RK4 class

version: 1.7

description:

A fourth order Runge-Kutta IODE solver. Constructs the RK4 IODESolver for a system of ordinary differential equations.

usage:

var inst:RK4 = new RK4(_ode);

parameters:

  • _ode (IODE) -- the system of differential equations.

com.wis.math.calc.solvers.RK4 Properties:

com.wis.math.calc.solvers.RK4 Methods:

initialize

description: Initializes the IODE solver. Four rate arrays and three temporary state arrays are allocated. The number of differential equations is determined by invoking getState().length on the IODE.

usage: 

inst.initialize(_stepSize)

parameters:

  • _stepSize (Number) -- step integer.

returns: (Void)

step

description: Steps (advances) the differential equations by the stepSize. The IODESolver invokes the IODE's getRate method to obtain the initial state of the system. The IODESolver then advances the solution and copies the new state into the state array at the end of the solution step.

usage: 

inst.step()

returns: (Number) -- returns step size.

setStepSize

description: Sets the step size. The step size remains fixed in this algorithm.

usage: 

inst.setStepSize(_stepSize)

parameters:

  • _stepSize (Number) -- step integer.

returns: (Void)

getStepSize

description: Gets the step size. The stepsize is constant in this algorithm.

usage:

returns: (Number) -- returns step size.
