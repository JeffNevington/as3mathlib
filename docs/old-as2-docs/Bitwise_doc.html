com.wis.math.alg.Bitwise class

version: 1.6

description:

Implements the static behaviours of the Bitwise Class.

All methods in this class are static, allowing for inline bitwise operations without the need for class instantiation.

usage:

Bitwise.classMethod(args);

parameters:

  • none -- no class input parameters.

binToDec

description: Converts binary string input to decimal number output.

usage: 

Bitwise.binToDec("-01001");

parameters:

  • bin (String) -- must be a string to avoid octal conversion.

returns: (Number) -- returns a decimal base.

decToBin

description: Converts decimal number input to binary string output.

usage: 

Bitwise.decToBin(-9);

parameters:

  • dec (Number) -- positive or negative integer - method ignores fractional input.

returns: (String) -- returns a string representation of a binary base.

binPad

description: Pads binary string input to desired bit length string output.

usage: 

Bitwise.binPad("0111",32);

parameters:

  • bin (String) -- must be a string to avoid octal conversion.
  • len (Number) -- 2^n (up to 64) - usually 8, 32, or 64.

returns: (String) -- returns a string representation of a padded binary base.
