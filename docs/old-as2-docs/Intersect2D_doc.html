com.wis.math.geom.intersect2D.Intersect2D class

version: 1.7

description:

Implements the behaviours of the Intersect2D Class.

This class has been spread across the entire contents of the intersect2D package, due to the large byte size of the original JS document. Flash MX restricts class files to 32 k, so I have presented this class as the superclass to connect all of the 2D intersection routines.

usage:

var inst:Intersect2D = new Intersect2D(status);

parameters:

  • status (String) -- an identifier string.
