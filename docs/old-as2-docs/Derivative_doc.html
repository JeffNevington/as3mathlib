romberg

description: Calculates the derivative using the Romberg scheme for Richardson extrapolation.

This method runs until all Romberg rows are filled or until the step size drops below defaultNumericalPrecision or if the desired tolerance is reached.

usage: 

inst.romberg(f,x0,h,tol);

parameters:

  • f (Function) -- the function.
  • x (Number) -- where derivative is to be calculated.
  • h (Number) -- initial step size.
  • tol (Number) -- desired accuracy.

returns: (Number) -- returns the derivative.
