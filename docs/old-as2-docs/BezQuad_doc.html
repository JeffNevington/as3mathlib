com.wis.math.geom.curve.BezQuad class

version: 1.6

description:

Implements the behaviours of the BezQuad Class.

Supports display of my quadratic Bezier curve testing application's methodology. Presently, it only supports 2D -- I plan to re-work the architecture to support 3D definitons, as well as re-factoring for a more generic class instantiation.

usage:

var inst:BezQuad = new BezQuad();

parameters:

  • none -- no class input parameters.
