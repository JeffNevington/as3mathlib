com.wis.math.geom.trig.Circular class

version: 1.6

description:

Implements the static behaviours of the Circular Class.

Provides methods for the circular trigonometric functions that are not included in the core Math object.

usage:

Circular.circMethod(circParam);

parameters:

  • none -- no class input parameters.
