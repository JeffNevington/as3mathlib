com.wis.math.geom.util.Camera class

version: 1.7

description:

Implements the behaviours of the Camera Class.

Provides utility methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Camera = new Camera(loc,dir,right,down);

parameters:

  • loc (Vector) -- a position Vector object.
  • dir (Vector) -- a direction Vector object.
  • right (Vector) -- a direction Vector object.
  • down (Vector) -- a direction Vector object.
