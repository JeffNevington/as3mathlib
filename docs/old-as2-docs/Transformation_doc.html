com.wis.math.geom.util.Transformation class

version: 1.7

description:

Implements the behaviours of the Transformation Class.

Provides utility methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Transformation = new Transformation(vx,vy,vz,c[,dontFindInverse,actualOrder]);

parameters:

  • vx (Vector) -- an x-axis Vector object.
  • vy (Vector) -- a y-axis Vector object.
  • vz (Vector) -- a z-axis Vector object.
  • c (Vector) -- a translation Vector object.
  • dontFindInverse (Boolean) -- if 'false', initialization populates $inverse Transformation object.
  • actualOrder (Boolean) -- if 'true', copies passed vx, vy, and vz directly, else creates new Vector objects from their elements.
