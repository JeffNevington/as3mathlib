com.wis.math.alg.Matrix class

version: 1.6

description:

Implements the behaviours of the Matrix Class -- provides methods for matrix manipulation.

usage:

var inst:Matrix = new Matrix(r,c)

parameters:

  • r (Number) -- number of rows in new matrix.
  • c (Number) -- number of columns in new matrix.

-- number of columns in matrix.
