com.wis.math.geom.util.Noise class

version: 1.7

description:

Implements the behaviours of the Noise Class.

Possible input values are (.8296815753293, 11). Don't go high with the second argument; it's exponential. Creation of a Noise object can be very slow; evaluation of one isn't all that speedy either. This class includes duplication of method strategy to support existing applications.

usage:

var inst:Noise = new Noise(seedNum,power);

parameters:

  • seedNum (Number) -- a real number between 0 and 1.
  • power (Number) -- creates a multiple of 2.

com.wis.math.geom.util.Noise Properties:
