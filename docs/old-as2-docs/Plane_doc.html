com.wis.math.geom.polytope.Plane class

version: 1.7

description:

Implements the behaviours of the Plane Class.

Provides methods for the IObj interface based on JS RayTracer2 by John Haggerty. Since the interface doesn't permit overloading methods, I've added the 'getNormalAt' method that doesn't pass parameters, which retrieves the normal ($norm) to this instance, and I've changed 'getNormalAtUntransformed' into an abstract method that supports the interface structure but doesn't get called. This is a workaround and may change with testing.

usage:

var inst:Plane = new Plane(normal,dist);

parameters:

  • normal (Vector) -- a Vector object normal to this plane.
  • dist (Number) -- a scalar value.
