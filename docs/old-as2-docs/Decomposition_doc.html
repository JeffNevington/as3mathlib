com.wis.math.alg.Decomposition class

version: 1.7

description:

Implements the behaviours of the Decomposition Class. Provides methods for calculating matrix decomposition.

usage:

var inst:Decomposition = new Decomposition(max);

parameters:

  • max (Number) -- positive integer to clamp maximum newton iterations.

factorCholesky

description: Factors the symmetric positive definite matrix A in the form A = LDU, where L is the transpose of U, a unit upper-triangular matrix, and where D is a diagonal matrix. A square root-free modification of Cholesky's method is used.

usage: 

inst.factorCholesky();

returns: (Boolean)

newton

description: Newton's method with damping in n-space.

An attempt to solve f(x) = 0 using Newton's method with damping. The Jacobian matrix of f is approximated using finite differences.

usage: 

inst.newton(m,f,p_arr,bound);

parameters:

  • m (Number) -- positive integer for square matrix order.
  • f (Function) -- the function.
  • p_arr (Array) -- array to hold initial vector.
  • bound (Number) -- error bound.

returns: (Array) -- returns solution set.
