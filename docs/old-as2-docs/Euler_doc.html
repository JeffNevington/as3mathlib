com.wis.math.calc.solvers.Euler class

version: 1.7

description:

Euler implements the behaviours of the IODESolver Interface -- it constructs the Euler IODESolver for a system of ordinary differential equations. The Euler method is unstable for many systems. It is included as an example of how to use the IODE and IODESolver interface.

usage:

var inst:Euler = new Euler(_ode);

parameters:

  • _ode (IODE) -- the system of differential equations.

com.wis.math.calc.solvers.Euler Properties:

com.wis.math.calc.solvers.Euler Methods:

initialize

description: Initializes the IODE solver. The rate array is allocated. The number of differential equations is determined by invoking getState().length on the IODE.

usage: 

inst.initialize(_stepSize);

parameters:

  • _stepSize (Number) -- step integer.

returns: (Void)

step

description: Steps (advances) the differential equations by the $stepSize. The IODESolver invokes the IODE's getRate method to obtain the initial state of the system. The IODESolver then advances the solution and copies the new state into the state array at the end of the solution step.

usage: 

inst.step();

returns: (Number) -- returns step size.

setStepSize

description: The step size remains fixed in this algorithm.

usage: 

inst.setStepSize(_stepSize);

parameters:

  • _stepSize (Number) -- step integer.

returns: (Void)

getStepSize

description: The stepsize is constant in this algorithm.

usage:

returns: (Number) -- returns the step size.
