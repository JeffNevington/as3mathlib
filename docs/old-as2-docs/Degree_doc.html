com.wis.math.geom.trig.Degree class

version: 1.6

description:

Implements the static behaviours of the Degree Class.

Provides methods for calculating the main circular trigonometric functions by passing the angle input in degrees for sin, cos, and tan, and returning degrees for asin, acos, atan, and atan2.

usage:

Degree.classMethod(n);

parameters:

  • none -- no class input parameters.
