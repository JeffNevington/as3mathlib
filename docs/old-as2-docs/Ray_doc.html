com.wis.math.geom.util.Ray class

version: 1.6

description:

Implements the behaviours of the Ray Class.

Provides methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Ray = new Ray(start,dir,distMult);

parameters:

  • start (Vector) -- a position Vector object.
  • dir (Vector) -- a direction Vector object.
  • distMult (Number) -- a scalar value for this instance's magnitude.
