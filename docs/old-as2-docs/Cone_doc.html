com.wis.math.geom.quadric.Cone class

version: 1.7

description:

Implements the behaviours of the Cone Class.

Provides methods for the IObj interface based on JS RayTracer2 by John Haggerty. Since the interface doesn't permit overloading methods, I've added the 'getNormalAt' method to define the normal to the passed intersection and 'pos' Vector object, and I've changed 'getNormalAtUntransformed' into an abstract method that supports the interface but doesn't get called. This is a workaround and may change with testing.

usage:

var inst:Cone = new Cone(end1,rad1,end2,rad2,open);

parameters:

  • end1 (Vector) -- a position Vector object defining the center of one end of the cone.
  • rad1 (Number) -- a value for radius of one end of the cone.
  • end2 (Vector) -- a position Vector object defining the center of the other end of the cone.
  • rad2 (Number) -- a value for radius of the other end of the cone.
  • open (Boolean) -- default is 'false', cone ends are closed.
