com.wis.math.calc.solvers.RK45 class

version: 1.7

description:

RK45 implements the behaviours of the IODESolver Interface -- a Runge-Kutta 4/5 IODE solver with variable step size. Constructs the RK45 IODESolver for a system of ordinary differential equations.

usage:

var inst:RK45 = new RK45(_ode);

parameters:

  • _ode (IODE) -- the system of differential equations.

com.wis.math.calc.solvers.RK45 Methods:

initialize

description: Initializes the IODE solver. Temporary state and rate arrays are allocated. The number of differential equations is determined by invoking getState().length on the IODE.

usage: 

inst.initialize(_stepSize);

parameters:

  • _stepSize (Number) -- the step size.

returns: Void.

step

description: Steps (advances) the differential equations by the $stepSize. The IODESolver invokes the IODE's getRate method to obtain the initial state of the system. The IODESolver then advances the solution and copies the new state into the state array at the end of the solution step.

usage: 

inst.step();

returns: (Number) -- returns the step size.

setStepSize

description: Sets the step size. The step size may change when the step method is invoked.

usage: 

inst.setStepSize(_stepSize);

parameters:

  • _stepSize (Number) -- the step size.

returns: Void.

getStepSize

description: Gets the step size. The stepsize is adaptive and may change as the step() method is invoked.

usage: 

inst.getStepSize();

returns: (Number) -- returns the step size.
