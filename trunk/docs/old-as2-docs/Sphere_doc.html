com.wis.math.geom.quadric.Sphere class

version: 1.7

description:

Implements the behaviours of the Sphere Class.

Provides methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Sphere= new Sphere(pos,rad);

parameters:

  • pos (Vector) -- a position Vector object.
  • rad (Number) -- a real number for radius value.
