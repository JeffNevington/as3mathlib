com.wis.math.geom.util.Intersection class

version: 1.6

description:

Implements the behaviours of the Intersection Class.

Provides methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Intersection = new Intersection(depth,ray,obj);

parameters:

  • depth (Number) -- a scalar value.
  • ray (Ray) -- a Ray object.
  • obj (Object) -- first object passed to stack.
