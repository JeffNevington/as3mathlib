com.wis.types.lists.Dictionary class

version: 1.6

description:

Implements the behaviours of the Dictionary Class.

Provides methods for creatung a lookup dictionary, similar in concept to an associative array.

usage:

var inst:Dictionary = new Dictionary(keyList,dataList)

parameters:

  • keyList (Array) -- a list of categories.
  • dataList (Array) -- a list of category data.
