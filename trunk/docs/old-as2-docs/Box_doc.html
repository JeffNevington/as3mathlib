com.wis.math.geom.polytope.Box class

version: 1.7

description:

Implements the behaviours of the Box Class.

Provides methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Box = new Box(v1,v2,isBound);

parameters:

  • v1 (Vector) -- a position Vector object defining one corner of the box.
  • v2 (Vector) -- a position Vector object defining the opposite corner of the box.
  • isBound (Boolean) -- if 'true', leaves dimensions as is, else inverses dimensions.

com.wis.math.geom.polytope.Box Properties:

$v1

(Vector) -- a position Vector object defining one corner of the box.

$v2

(Vector) -- a position Vector object defining the opposite corner of the box.

$dimsInv

(Vector) -- if 'isBound' is 'true', an identity Vector object, else adds $v2 to the inverse of the negative of $v1.

$negV1

(Vector) -- a position Vector object defining negative of $v1.

$boundedBy

(Box) -- a bounding box object.
