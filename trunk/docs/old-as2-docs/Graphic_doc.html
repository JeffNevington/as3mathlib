Graphic.intersectLines(l1,l2);

parameters:

  • l1 (Object) -- line object containing 2 Point objects.
  • l2 (Object) -- line object containing 2 Point objects.

returns: (Point) -- returns a Point object for the line intersection point.

intersectLinesImplicit

description: Defines an intersection point of two lines in 2-space using the implicit line equation: 

     f(x,y) = ax + by + c = 0
Expects to be passed two: line object = {p1:Point,p2:Point}

usage: 

     Graphic.intersectLinesImplicit(l1,l2);
     // which in turn calls:
     var t1:Intersect2DLine = new Intersect2DLine();
     var t2:Intersect2D = t1.lineLine(l1.p1,l1.p2,l2.p1,l2.p2);
     var pt:Point = new Point(t2.$points[0].x,t2.$points[0].y);

parameters:

  • l1 (Object) -- line object containing 2 Point objects.
  • l2 (Object) -- line object containing 2 Point objects.

returns: (Point) -- returns a Point object for the line intersection point.

intersectLinesParametric

description: Defines in 3-space either an intersection point of two lines, or else the more common case, the shortest path to the nearest point of each line, using the parametric line equation: 

     P(t) = P0 + tvL
Expects to be passed two: line object = {p1:Point,p2:Point}

usage: 

Graphic.intersectLinesParametric(l1,l2[,true]);

parameters:

  • l1 (Object) -- line object containing 2 Point objects.
  • l2 (Object) -- line object containing 2 Point objects.
  • bSeg (Boolean) -- clamps calculation to line segments if passed true.

returns: (Point) -- returns a Point object for the line intersection point.
