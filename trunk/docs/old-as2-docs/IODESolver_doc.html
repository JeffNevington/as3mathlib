com.wis.math.calc.ode.IODESolver interface

version: 1.7

description:

IODESolver interface defines a minimal differential equation solver.

usage:

var inst:IODESolver = new IODESolver();

parameters:

  • none -- interface has no parameters.

com.wis.math.calc.ode.IODESolver Methods:

initialize

description: Initializes the IODE solver. IODE solvers use this method to allocate temporary arrays that may be required to carry out the solution. The number of differential equations is determined by invoking getState().length on the ODE.

usage: 

inst.initialize(_stepSize);

parameters:

  • _stepSize (Number) -- the step size.

returns: (Void)

step

description: Steps (advances) the differential equations by the stepSize. The IODESolver invokes the IODE's getRate method to obtain the initial state of the system. The IODESolver then advances the solution and copies the new state into the state array at the end of the solution step.

usage: 

inst.step();

returns: -- returns the step size.

setStepSize

description: Sets the initial step size. The step size may change if the IODE solver implements an adaptive step size algorithm such as RK4/5.

usage: 

inst.setStepSize(_stepSize);

parameters:

  • _stepSize (Number) -- the step size.

returns: (Void)

getStepSize

description: Gets the step size.

usage: 

inst.getStepSize(();

returns: (Number) -- returns the step size.
