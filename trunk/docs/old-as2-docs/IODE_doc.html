com.wis.math.calc.ode.IODE interface

version: 1.7

description:

IODE defines a system of differential equations by providing access to the rate equations.

usage:

var inst:IODE = new IODE();

parameters:

  • none -- interface has no parameters.

com.wis.math.calc.ode.IODE Methods:

getState

description: Gets the state variables. The getState method is invoked by an ODESolver to obtain the initial state of the system. The ODE solver advances the solution and then copies new values into the state array at the end of the solution step.

usage: 

inst.getState();

returns: state (Array) -- returns the state array.

getRate

description: Gets the rate of change using the argument's state variables. This method may be invoked many times with different intermediate states as an ODESolver is carrying out the solution.

usage: 

inst.getRate(state,rate);

parameters:

  • state (Array) -- the state array.
  • rate (Array) -- the rate array.

returns: Void.
