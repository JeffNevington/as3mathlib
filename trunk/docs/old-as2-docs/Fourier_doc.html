com.wis.math.calc.Fourier Methods:

dft

description: Discrete Fourier transform -- this calculates an in-place complex-to-complex dft. x_arr and y_arr are the real and imaginary number arrays of 2^m points. This algorithm is about ten times slower than fft, but is useful when comparing many transforms.

usage: 

Fourier.dft(dir,m,x_arr,y_arr);

parameters:

  • dir (Number) -- 1 gives forward transform, -1 gives reverse transform.
  • m (Number) -- a positive integer.
  • x_arr (Array) -- an array containing x-axis values for real number input.
  • y_arr (Array) -- an array containing y-axis values for imaginary number input.

returns: (Boolean)

fft

description: Fast Fourier transform -- this calculates an in-place complex-to-complex fft. x_arr and y_arr are the real and imaginary number arrays of 2^m points. 

     Formula: forward
                 N-1
                 ---
             1   \          - j k 2 pi n / N
     X(n) = ---   >   x(k) e                    = forward transform
             N   /                                n=0..N-1
                 ---
                 k=0

     Formula: reverse
                 N-1
                 ---
                 \          j k 2 pi n / N
     X(n) =       >   x(k) e                    = reverse transform
                 /                                n=0..N-1
                 ---
                 k=0

usage: 

Fourier.fft(dir,m,x_arr,y_arr);

parameters:

  • dir (Number) -- 1 gives forward transform, -1 gives reverse transform.
  • m (Number) -- a positive integer.
  • x_arr (Array) -- an array containing x-axis values for real number input.
  • y_arr (Array) -- an array containing y-axis values for imaginary number input.

returns: (Boolean)

fft2D

description: Performs a 2D fft in place given a complex 2D array.

usage: 

Fourier.fft2D(comp_arr,nx,ny,dir);

parameters:

  • comp_arr (Array) -- a 2d array, each element contains a {re:val,im:val} Complex object.
  • nx (Number) -- an integer, the size of the array rows.
  • ny (Number) -- an integer, the size of the array columns.
  • dir (Number) -- the direction dir, 1 for forward, -1 for reverse.

returns: (Boolean) -- returns false if there are memory problems or the dimensions are not powers of 2
