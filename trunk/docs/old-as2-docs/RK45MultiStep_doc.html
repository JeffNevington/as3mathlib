com.wis.math.calc.solvers.RK45MultiStep class

version: 1.7

description:

RK45MultiStep extends RK45, which implements the behaviours of the IODESolver Interface. Perform multiple Runge-Kutta 4/5 IODE steps so that a uniform step size is maintained. Constructs the RK45MultiStep IODESolver for a system of ordinary differential equations.

usage:

var inst:RK45MultiStep = new RK45MultiStep(_ode);

parameters:

  • _ode (IODE) -- the system of differential equations.

com.wis.math.calc.solvers.RK45MultiStep Methods:

initialize

description: Initializes the IODE solver. Temporary state and rate arrays are allocated by invoking the superclass method.

usage: 

inst.initialize(_stepSize);

parameters:

  • _stepSize (Number) -- the step size.

returns: Void.

step

description: Steps (advances) the differential equations by the stepSize. The IODESolver invokes the IODE's getRate method to obtain the initial state of the system. The IODESolver then advances the solution and copies the new state into the state array at the end of the solution step.

usage: 

inst.step();

returns: (Number) -- returns the step size.

setStepSize

description: Sets the step size.

usage: 

inst.initialize(_stepSize);

parameters:

  • _stepSize (Number) -- the step size.

returns: Void.

getStepSize

description: Gets the step size. The step size is the fixed step size, not the size of the RK4/5 steps that are combined into a single step.

usage: 

inst.getStepSize();

returns: (Number) -- returns the step size.

plus

description: Steps the ode with a positive stepsize.

usage: 

inst.plus();

returns: (Number) -- returns the step size.

minus

description: Steps the $ode with a negative stepsize.

usage: 

inst.minus();

returns: (Number) -- returns the step size.
