com.wis.math.geom.util.Finish class

version: 1.7

description:

Implements the behaviours of the Finish Class.

Provides utility methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Finish = new Finish(specular,glossiness,reflection,ambient,diffuse)

parameters:

  • specular (Number) -- a real number between 0 and 1.
  • glossiness (Number) -- a real number between 1 and infinity.
  • reflection (Number) -- a real number between 0 and 1.
  • ambient (Number) -- a real number between 0 and 1.
  • diffuse (Number) -- a real number between 0 and 1.
