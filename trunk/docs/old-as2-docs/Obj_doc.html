com.wis.types.Obj class

version: 1.6

description:

Implements the behaviours of the Obj SuperClass.

Provides methods for the IObj interface based on JS RayTracer2 by John Haggerty.

usage:

var inst:Obj = new Obj(arguments);

parameters:

  • arguments (variant) -- arguments vary, depending on subclass instantiation.

getNormalAtUntransformed

description: Virtual function to be overridden by subclass ... which defines bounding box to vector normal in 3-space before transformation.

usage:

parameters:

  • pos (Vector) -- a position Vector object.

returns: (Vector) -- returns normal of untransformed vector.

getColorAt

description:

usage: 

inst.getColorAt(isect,objs,lights,traceLevel,effect);

parameters:

  • isect (Intersection) -- focus Intersection class instance.
  • objs (Array) -- a list of shape objects passed to a Ray class instance.
  • lights (Array) -- a list of LightSource class instances.
  • traceLevel (Number) -- a real number passed to test against class instance $maxTraceLevel property.
  • effect (Number) -- a real number passed to test against class instance $bailout property.

returns: (Col) -- returns a Col object for this subclass instance.

returns: (Number) -- returns a positive or negative value which is evaluated by the subclass' Array.sort method.

getNormalAt

description: Defines normal vector based on the 'pos' Vector object and the focus intersection using a winding number algorithm affecting the direction of the class instance $transform property.

usage: 

inst.getNormalAt(pos,isect);

parameters:

  • pos (Vector) -- an object defining a point in 3-space wrt the subclass instance.
  • isect (Intersection) -- focus intersection.

returns: (Vector) -- returns a normalized Vector object.

usage:

parameters:

returns: (Boolean)

copyModifiers

description: Copies subclass object's superclass properties to 'newObj' argument and calls the new class instance 'initialize' method.

usage:

parameters:

returns: (Obj) -- returns a new Obj class instance with a copy of inherited properties.
