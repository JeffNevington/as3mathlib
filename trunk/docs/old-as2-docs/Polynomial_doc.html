com.wis.math.alg.Polynomial class

version: 1.6

description:

Implements the behaviours of the Polynomial Class -- provides methods for polynomial calculations up to the 4th degree.

usage:

var inst:Polynomial = new Polynomial()

parameters:

  • none -- no class input parameters.

com.wis.math.alg.Polynomial Properties:

$coefs

-- array to hold polynomial coefficients.

$TOLERANCE

-- static -- minimum value to truncate computation.

$ACCURACY

-- static -- ceiling applied to log value in 'bisection' method.
